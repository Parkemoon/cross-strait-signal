# Portrait Image Attributions

All portrait images in this directory are sourced from Wikimedia Commons under public domain or Creative Commons licences.

To add a portrait:
1. Download from the Wikimedia Commons URL below (or a suitable alternative)
2. Crop/resize to approximately 200×200px, centred on the face
3. Save as the filename listed in `key_figures.json` (e.g. `xi_jinping.jpg`)
4. Add the source URL and licence to this file
